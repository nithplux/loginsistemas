git iniciar repositorio -->git init
git colocar nombre de usuario --> git config user.name "tu apodo (puede ser cualqiera)"
git colocar tu correo --> git config user.email "tu correo de github"
git clone "colocas la direccion url de tu proyecto" -->clonar el repositorio main (principal)
git pull --> actualizar archivos
git fetch -->actualizar configuraciones
git branch -->listar las ramas
git checkout -b "nombre de la rama" -->crear rama y seleccionarla
git log -->para visualizar los movimientos y modificaciones que hicieron los usuarios
git add --> enviar tus archivos modificados o creados para tu stage area
git push -u origin "nombre de la rama"-->para actualizar los archivos modificados o agregar nuevos o se actualizan remotamente las ramas
git commit -m "texto que desees version 1...." -->realizar un commit
git branch -D nombre-rama --> eliminar rama de repositorio local
git push origin :nombre-rama -->eliminar rama de repositorio remoto (tener cuidado con el espacio que se deja entre origin :nombre-rama
ya que se tiene que dejar el espacio entre origin y los dos puntos porque sino no funcionara.
git branch -m developer -->cambiar nombre a la rama
